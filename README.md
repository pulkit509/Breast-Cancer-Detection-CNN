# Breast Cancer Detection using CNN

This project detects breast cancer from ultrasound images using deep learning.

## Dataset
Breast Ultrasound Images Dataset  
https://www.kaggle.com/datasets/aryashah2k/breast-ultrasound-images-dataset

## Model
Transfer Learning using MobileNetV2

## Classes
- Normal
- Benign
- Malignant

## Pipeline
Dataset → Preprocessing → Data Augmentation → CNN Training → Evaluation

## Technologies
- Python
- TensorFlow / Keras
- OpenCV
- Google Colab
